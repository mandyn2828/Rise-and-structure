# Onboarding Flow — 5-Step Assessment

## Overview
A 5-step onboarding experience that begins with understanding the user's starting point, then reveals a personalized plan. This replaces the tier-selection-first approach — now we assess first, then prescribe.

## Flow Diagram
```
Step 1: Welcome & Assessment → Step 2: Wellness Baseline → Step 3: Mindset Check-In → Step 4: Finance Goals → Step 5: Personalized Plan Reveal
```

## User Data Collected
By the end of Step 5, we know:
- Primary motivation(s) for joining
- Wellness starting point (1-5 scale per area)
- Current stress/mental state
- Financial situation & income goals
- Time availability per day
- Preferred delivery time

This data drives the personalized plan.
