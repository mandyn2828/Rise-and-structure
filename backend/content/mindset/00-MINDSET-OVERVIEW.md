# Mindset Coaching — Overview

## Philosophy
Mid-life is a second adolescence, emotionally speaking. Old dreams collide with current reality. Regret, comparison, and "is this it?" thoughts surface regularly. Mindset coaching doesn't eliminate these — it gives the user tools to process them and turn them into fuel.

## Weekly Structure

| Day | Theme | Focus |
|-----|-------|-------|
| Mon | Identity Shift | "Who am I becoming?" |
| Tue | Self-Compassion | Quieting the inner critic |
| Wed | Purpose Reconnect | What matters NOW |
| Thu | Emotional Awareness | Name it to tame it |
| Fri | Resilience | Reframing struggle |
| Sat | Connection | Relationship with self & others |
| Sun | Vision | Looking forward with intention |

## Format per Day
1. **Prompt** (1-2 sentences to start the day)
2. **Reflection** (questions to journal on)
3. **Practice** (actionable exercise)
4. **Affirmation** (repeatable phrase)
5. **Evening Check-in** (closing reflection)

## Delivery
- Morning: prompt + reflection questions (sent via app notification/email)
- Anytime: practice (self-guided)
- Evening: check-in (closes the loop)
- Weekend versions are lighter
