# Day 1 — Identity Shift

## Morning Prompt
> "You are not who you were at 25. Who are you becoming now?"

## Reflection (Write for 5 min)

1. What's one belief I held at 25 that I no longer believe?
2. What's one thing I want people to say about me 5 years from now?
3. If I removed "should" from my vocabulary for one week, what would change?
4. What identity am I clinging to that no longer serves me? ("I'm not a morning person." "I'm bad with money." "I'm not the type who exercises.")
5. Who do I secretly want to be? (Be honest — no one's reading this.)

## Practice: The Identity Audit

Look at these identity pillars. Rate each 1-10 on how aligned you feel:

| Pillar | Current (1-10) | Desired (1-10) | One small change to close the gap |
|--------|---------------|----------------|-----------------------------------|
| Physical health | | | |
| Financial security | | | |
| Career / Purpose | | | |
| Relationships | | | |
| Personal growth | | | |

**Today's Action:** Pick ONE pillar where the gap is largest. Do one thing today (5 min or less) that moves you toward the "desired" version.

## Affirmation (Repeat 3x today)
> "I am in the process of becoming. Who I was is not who I will be."

## Evening Check-in (2 min)
> "Did I act in alignment with who I want to become today? What would I do differently tomorrow?"

## Coach's Note
Identity isn't found — it's built. Every small choice is a vote for the person you're becoming. Cast votes wisely.
