# Rise & Structure — User-Facing Docs Overview

**Files in this directory:**

| File | Purpose | Audience |
|------|---------|----------|
| `FAQ.md` | In-app FAQ / help guide — answers the top 20+ questions | All users |
| `quickstart-guide.md` | Concise getting-started guide for Day 0-7 | New members |
| `welcome-page.md` | Welcome page content shown after sign-up | New sign-ups |

These docs are designed to reduce support requests and answer the questions mid-lifers actually have before they ask.

## Tone Notes for All Docs
- **Direct and warm** — like a smart friend who's been through it
- **No jargon** — "Wellness routine" not "somatotropic recovery protocol"
- **Short paragraphs** — mid-lifers are skimming, not studying
- **Action-oriented** — every answer ends with what to do next
